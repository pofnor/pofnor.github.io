Regular Expression Syntax :

/pattern/modifier(s)

Modifier i = Perform case-insensitive matching
Modifier g = Perform a global match (find all matches rather than stopping after the first match)
Modifier m = Perform multiline matching
[] are used to find a range of characters:
d = Find a digit
D = Find a non-digit
w = Find a word character
W = Find a non-word character
s = Find a whitespace character
$ = Matches any string at the end of it
^ = Matches any string at the beginning of it
+ = Matches any string that contains at least one
* = Matches any string that contains zero or more
? = Matches any string that contains zero or one
. = matches any character, except newline or other line terminators.
(?! ) = Negative lookahead. Specifies a group that can not match after the main expression (if it matches, the result is discarded).
Like /inp+(?!u)/g that find every "inp" which is not followed by "u"
Like /^((?!badword).)*$/gm that Match string not containing "badword"
